# Campus News Retrieval System

### 介绍
本项目是针对深圳大学公文通的一个检索系统，内含爬虫，先把信息爬下来，处理，然后导入检索系统进行初始化，之后便可以实现检索。自动爬虫更新后台信息尚未实现。

### 准备工作
* 在eclipse中导入本项目，运行
